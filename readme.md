# PLT React-Native Mobile App ( iOS)

## Running the app locally

```console
npm install
```

Then use the following commands to run iOS or Android:

```console
npm run ios
```
---

## Troubleshooting

* Clean ios build:

    ```console
    cd ios
    rm -rf build/
    pod install
    ```

    or

    ```console
    pod update
    ```
